Service /api/maths - Documentation

Ce service api donne l'option de faire:

  • Une addition
  • Une soustraction
  • Une multiplication
  • Une division
  • Le calcul du reste
  • La factorielle d'un nombre entier
  • La vérification d'un nombre premier
  • Le nième nombre premier

Dépendament de l'opération choisit, vous auriez à utiliser certains de ses paramètres:
op (qui spécifie l'opération demandez)
x (qui servira du premier chiffre)
y (qui servira du deuxième chiffre)
n (qui servira pour les opérations qui font rapport à un entier premier)

Pour faire une requête, il faudra ajouter à l'URL les paramètres demandés d'une certaine façon.
Par défaut l'URL sans opération sera celle-ci: https://201969281-api-server.glitch.me/api/maths?
N'oubliez pas les "&" entre les paramètres, sinon le service ne fonctionnera pas.

Pour l'addition (+):

Il vous faut spécifier l'opération (op), le chiffre x et le chiffre y, par exemple si vous vouliez faire 20+5, vous feriez:
https://201969281-api-server.glitch.me/api/maths?op=+&x=20&y=5

Pour la soustraction (-):

Il vous faut spécifier l'opération (op), le chiffre x et le chiffre y, par exemple si vous vouliez faire 20-5, vous feriez:
https://201969281-api-server.glitch.me/api/maths?op=-&x=20&y=5

Pour la multiplication (*):

Il vous faut spécifier l'opération (op), le chiffre x et le chiffre y, par exemple si vous vouliez faire 20*5, vous feriez:
https://201969281-api-server.glitch.me/api/maths?op=*&x=20&y=5

Pour la division (/):

Il vous faut spécifier l'opération (op), le chiffre x et le chiffre y, par exemple si vous vouliez faire 20/5, vous feriez:
https://201969281-api-server.glitch.me/api/maths?op=/&x=20&y=5

Pour le calcul du reste (%):

Il vous faut spécifier l'opération (op), le chiffre x et le chiffre y, par exemple si vous vouliez faire 20%3, vous feriez:
https://201969281-api-server.glitch.me/api/maths?op=%&x=20&y=3

Pour la factorielle d'un nombre entier (!):

Il vous faut spécifier l'opération (op) et le nombre entier (n), par exemple si vous vouliez faire la factorielle de 5, vous feriez:
https://201969281-api-server.glitch.me/api/maths?op=!&n=5

Pour la vérification d'un nombre premier (p):

Il vous faut spécifier l'opération (op) et le nombre entier (n) par exemple si vous vouliez faire la vérification de 5, vous feriez:
https://201969281-api-server.glitch.me/api/maths?op=p&n=5

Pour le nième nombre premier (np):

Il vous faut spécifier l'opération (op) et le nombre entier (n) par exemple si vous vouliez trouver le 5e nombre premier, vous feriez:
https://201969281-api-server.glitch.me/api/maths?op=np&n=5
